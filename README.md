# Java Array Programs - Day 4 Work

A collection of Java programs demonstrating various array operations including 1D arrays, 2D matrices, and common array algorithms.

## 📋 Programs Included

### 1D Array Programs
| # | Program | Filename | Description |
|---|---------|----------|-------------|
| 1 | Sum of Elements | `SumOfElements.java` | Calculate sum of all array elements |
| 2 | Max and Min | `MaxMin.java` | Find largest and smallest element in array |
| 3 | Even/Odd Count | `EvenOddCount.java` | Count even and odd numbers in array |
| 4 | Reverse Array | `ReverseArray.java` | Print array elements in reverse order |
| 5 | Linear Search | `LinearSearch.java` | Search for an element in array |

### 2D Array (Matrix) Programs
| # | Program | Filename | Description |
|---|---------|----------|-------------|
| 6 | Print Matrix | `PrintMatrix.java` | Display 2D array in grid format |
| 7 | Sum 2D | `Sum2D.java` | Calculate sum of all matrix elements |
| 8 | Row Sum | `RowSum.java` | Calculate sum of each row separately |
| 9 | Transpose | `TransposeMatrix.java` | Find transpose of a matrix |
| 10 | Diagonal Elements | `DiagonalElements.java` | Print primary and secondary diagonals |
